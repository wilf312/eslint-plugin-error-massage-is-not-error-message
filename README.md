# eslint-plugin-error-massage-is-not-error-message

## install
```
npm i eslint-plugin-error-massage-is-not-error-message
```

## setting

```
{
  "plugins": [
    "eslint-plugin-error-massage-is-not-error-message"
  ],
  "rules": {
    "error-massage-is-not-error-message/error-massage": 2
  }
}
```
